Getting started using Jounce.

It's quite simple, really. All you need to do is follow the three steps listed below:

1. Visit the library to view our selection of the latest space documentaries.

2. Pick one you would like more information on, and note the number assigned to it.

3. Hover over "Get Movies" and pick the corresponding number.

Voila! Now you can do it all over again with another movie if you fancy so.
